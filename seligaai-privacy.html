Política de Privacidade

Última atualização: 06 de janeiro de 2026

1. Introdução

A VisionX ("nós", "nosso" ou "empresa") desenvolve e opera o aplicativo SE LIGA AÍ ("App", "Aplicativo" ou "Serviço"). Esta Política de Privacidade descreve de forma clara e transparente como as informações são tratadas ao utilizar o aplicativo.

O SE LIGA AÍ tem como objetivo auxiliar na segurança pessoal, alertando usuários sobre áreas com maior incidência de riscos nas cidades de Pedreiras e Trizidela do Vale – MA, com base em dados públicos e análises locais.

Ao utilizar o aplicativo, você concorda com os termos desta Política de Privacidade.

2. Informações que Coletamos

2.1 Dados de Localização

  • Localização atual: O aplicativo pode acessar sua localização geográfica (latitude e longitude)
  • Quando ocorre: Durante o uso do aplicativo e, quando autorizado, em segundo plano
  • Finalidade: Identificar proximidade com áreas de risco e emitir alertas de segurança
  • Precisão: Utilização de localização precisa (GPS)

Importante: A localização em segundo plano é utilizada apenas quando necessária para o funcionamento dos alertas, não sendo armazenado histórico de deslocamento.

2.2 Histórico de Alertas

  • Data e hora dos alertas recebidos
  • Tipo de alerta (ex.: área de risco, atenção, zona sensível)
  • Região aproximada onde o alerta foi disparado

Armazenamento: Exclusivamente local, no dispositivo do usuário

2.3 Identificador do Aplicativo

  • Identificador local gerado automaticamente no primeiro uso
  • Utilizado apenas para funcionamento interno do app
  • Não permite identificação pessoal do usuário

2.4 Dados Técnicos

  • Modelo do dispositivo
  • Sistema operacional e versão do aplicativo
  • Status de permissões concedidas

Esses dados são utilizados apenas para garantir o funcionamento adequado do aplicativo.

3. Como Utilizamos as Informações

As informações coletadas são utilizadas exclusivamente para:

  • Emitir alertas de segurança com base na localização atual
  • Exibir áreas de risco no mapa
  • Manter o histórico pessoal de alertas
  • Correção de falhas e melhorias técnicas

O que NÃO fazemos

  • Não vendemos dados pessoais
  • Não utilizamos dados para publicidade
  • Não criamos perfis de comportamento
  • Não rastreamos rotinas ou trajetos do usuário

4. Base Legal para o Tratamento de Dados

O tratamento de dados é realizado conforme a Lei Geral de Proteção de Dados (LGPD), com base em:

  • Consentimento do usuário, concedido ao aceitar as permissões do aplicativo
  • Interesse legítimo, relacionado à segurança pessoal
  • Cumprimento de obrigação legal, quando aplicável

5. Compartilhamento de Informações

5.1 Dados de Usuário

O SE LIGA AÍ não compartilha dados pessoais ou de localização com terceiros.

5.2 Exceções Legais

A divulgação de informações poderá ocorrer apenas quando:

  • Exigida por lei ou ordem judicial
  • Necessária para cumprir obrigações legais

Sempre respeitando os limites técnicos do aplicativo.

6. Armazenamento e Segurança

6.1 Armazenamento

  • Todos os dados são armazenados localmente no dispositivo
  • O aplicativo não utiliza servidores externos para dados pessoais

6.2 Segurança

  • Uso dos mecanismos de segurança do sistema Android
  • Isolamento de dados (sandbox)
  • Controle de acesso por permissões do sistema

6.3 Retenção

  • Histórico de alertas permanece até ser excluído pelo usuário
  • Não há armazenamento de histórico de localização

7. Direitos do Usuário

Você pode, a qualquer momento:

  • Visualizar o histórico de alertas
  • Excluir alertas individualmente ou em lote
  • Silenciar notificações temporária ou permanentemente
  • Revogar permissões de localização nas configurações do Android

Ao desinstalar o aplicativo, todos os dados locais são removidos automaticamente.

8. Privacidade de Crianças

O SE LIGA AÍ não é destinado a menores de 13 anos e não coleta intencionalmente dados de crianças.

9. Notificações

  • Emitidas automaticamente ao detectar proximidade com áreas de risco
  • Processamento realizado localmente no dispositivo
  • Não há envio de dados para servidores externos

10. Tecnologias de Rastreamento

O aplicativo não utiliza:

  • Cookies
  • SDKs de anúncios
  • Ferramentas de analytics de terceiros
  • Identificadores de publicidade

11. Transferência Internacional de Dados

Não há transferência internacional de dados, pois todas as informações permanecem armazenadas localmente.

12. Atualizações desta Política

Esta Política pode ser atualizada periodicamente. Mudanças relevantes serão comunicadas por:

  • Notificação no aplicativo
  • Atualização da data deste documento

13. Base de Dados de Áreas de Risco

Os pontos exibidos no aplicativo são definidos a partir de:

  • Informações públicas e dados abertos de segurança
  • Análises estatísticas locais
  • Relatos e registros públicos disponíveis

O SE LIGA AÍ não é um aplicativo oficial nem afiliado a órgãos governamentais ou policiais. Os alertas têm caráter informativo e preventivo, não substituindo cuidados pessoais.
